# guide-cn-official

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存在，可以直接通过浏览器访问查看。仓库不针对任何特定域名或网站，仅作为页面资源的集合存储与版本管理用途。

## 目录说明

```
.
├── index.html          # 仓库主页 / 导航页
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

目录结构可能随更新而变化，请以实际仓库内容为准。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可在本地直接打开，或部署至任意静态托管服务。
- 页面内容可能随时间更新或调整，历史版本可通过 Git 提交记录查看。
- 仓库不保证所有页面长期可用或内容不变，建议使用者自行备份所需版本。

## 维护说明

- 本仓库由维护者不定期更新，新增、修改或删除页面均会通过 commit 记录体现。
- 欢迎通过 Issues 提交问题或建议，但请注意本仓库不针对具体网站或服务，一般性建议将根据情况采纳。
- 不接受直接向仓库提交 Pull Request 修改页面内容，除非是明显的格式或链接修复。
